Hexo 之后续篇

2015年4月17日

前言：

上一篇文记录了我部署 Hexo 的过程，然后某人跟我说还可以把 Hexo 部署到 gitcafe 或者自己的 vps 上，这几天也发现在 github 打开页面实在是太慢（穿墙会快一些），看来得找个时间把他迁出去了（真折腾…）。

现在记录下继上篇文之后未完的操作,大概就是这几件事：

  • Hexo 常用操作及配置。
  • 解析域名。
  • 某些问题及解决办法。
