为 Hexo 博客添加 404 页面

2016年3月10日

前言

一直想为自己的 hexo 博客添加 404 页面来着。 Google 搜到的很多方法，貌似都不是特别完美。这里记录下自己的创建方法。

大致思路如下

  • 创建404.md页面
  • 通过编辑Front-matter来定义页面的布局，固定链接；
  • 编辑 nginx 配置文件

我这里选择了 404 公益。有两个选择：

  • 腾讯404
